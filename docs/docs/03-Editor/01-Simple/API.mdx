## Usage

#### type Model

```go
type Model[T any] struct {
}
```

A Model is a simple editor for handling simple token-based inputs without any
special parsing required

#### func  New

```go
func New[T any](options ...Option[T]) *Model[T]
```
New initializes a model

#### func (*Model[T]) AllTokens

```go
func (m *Model[T]) AllTokens() []editor.Token
```
AllTokens returns a list of all tokens from the user input including delimiter
tokens.
